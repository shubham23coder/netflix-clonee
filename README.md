This is a full-stack Netflix Clone project built using React.js for the frontend and Node.js with Express for the backend. The project was created as part of an internship assignment. It includes a responsive UI similar to Netflix with a home page, hero banner, movie and series cards displayed in rows, navigation bar, and search functionality. React Router is used for navigation and the frontend is structured using reusable components. The backend provides simple REST APIs including GET /movies to fetch all movies, GET /movies/:id to fetch a specific movie, and POST /login for mock authentication. The data is stored in a JSON file or in-memory array. The project is deployed with the frontend on Vercel or Netlify and the backend on Render.
